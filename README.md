## Featured Projects

### 1. Khnhom. Secure Link-in-Bio with KHQR Generator
**The Problem:** Many local vendors and content creators struggle with sharing payment information securely. Manual copy pasting of banking details causes friction and increases the risk of errors.

**The Solution:** A centralized platform to safely store links and generate standardized Bakong KHQR codes for instant payments.

* **Key Features:**
    * Dynamic Bakong KHQR generation.
    * Khmer language support.
    * Google Safe Browsing integration for link validation.
    * Google OAuth
    * Rate limiting and security headers to prevent abuse.
* **Tech:** React, Next.js, Node.js (Express), MongoDB, Redis, Zod, Passport.js.

### 2. Perfume E-Commerce Platform
**The Context:** A custom solution built for a real family business, tailored to the specific purchasing behaviors of the Cambodian market.

**The Engineering:** Built without frontend frameworks, to showcase pure JavaScript competency, utilizing a strict Vanilla JS MVC Architecture bundled with Vite. The backend is a robust Flask API secured with Cloudflare Turnstile and Rate Limiting.

**The Solution:** Unlike western e-commerce flow, this platform builds trust by bridging the gap between web browsing and direct communication.

* **Key Features:**
    * Full product catalog and shopping cart system.
    * A custom Model-View-Controller structure handling state and UI rendering without heavy dependencies.
    * **Telegram Integration:** Orders are dispatched directly to the seller via Telegram API for immediate processing.
    * Payment-on-delivery flow to increase conversion rates.
    * Admin workflow for inventory management.
    * SPA like behavior for product page navigation.
* **Tech:** Flask, PostgreSQL, SQLAlchemy, Pydantic.
